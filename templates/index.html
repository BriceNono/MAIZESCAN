    How the model works

    01

    Image input

    Leaf photo resized to 224×224 px, pixel values normalised to [0,1]

    →
    02

    VGG16 feature extraction

    13 convolutional layers in 5 blocks extract spatial disease features from the leaf texture

    →
    03

    Dense classification

    512→256 neuron layers with BatchNorm and Dropout map features to disease probabilities

    →
    04

    Softmax output

    4-neuron softmax layer outputs disease probability distribution summing to 100%
